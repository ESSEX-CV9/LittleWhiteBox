功能说明：

1. 渲染被```包裹的html、!DOCTYPE、script的代码块内容为交互式界面

2. 提供STscript(command)异步函数执行酒馆命令

3. 数据统计：每条消息添加大脑图标，点击查看统计

4. 循环任务：自动执行设定好的斜杠命令

5. 消息预览：查看AI接收到的完整消息内容

6. 模板编辑器：为角色设置输出模板，AI输出会立即用模板渲染

配图说明：

• 自动从AI消息中提取角色、服装、场景等关键词

功能说明：

1. 让你能为聊天角色创建动态模板，支持参数提取和自定义渲染，AI开始输出时立即显示模板结构

2. 使用正则表达式从AI输出中提取参数，默认格式：[参数名]内容[/参数名]，模板中使用[[参数名]]格式引用参数

3. 模板设置会随角色卡一起导出/导入

4. 更详细的介绍请参考⮕文檔

使用示例：

// 显示提示消息
await STscript('/echo 你好世界！');

// 创建变量
await STscript('/setvar key=天气 晴朗');
const weather = await STscript('/getvar 天气');

// 动态更新界面
document.querySelector('#status').textContent = `当前天气：${weather}`;

// 创建按钮
const button = document.createElement('button');
button.textContent = '点击我';
button.onclick = () => STscript('/xbqte task');
document.body.appendChild(button);
